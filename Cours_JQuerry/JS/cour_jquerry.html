Séléctioner en JQuery

Le coeur Jquery se trouve dans la selection 'éléments pour ensuite éxécuter des méthodes dessus.

$(selecteur) accepte en argument tou selecteurs CSS , JQuery nous renvoie ensuite une collection d'éléménts DOM dans un obj JQuery .

$(div) Séléctione tout les divs $('.titre') séléctionne toutes les "class='titre'

Cette enveloppe JQuery est celle qui donne accé à toute les fonctionnalitées de jquery

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher des éléments
  • Les animés

tous les élément sélectionnés en javascript pur sont des éléent DOM et tous ceux sélectionnés avec $("...") sont des élémentsdans une enveloppe JQuery

JQuery dispose de plusieurs méthod pour parcourir le DOM

  • children() parcourt les enfants
  • find()parcourt les descendant
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie le parent
  • eq() renvoie, parmis une liste d'éléménts, celui qui se trouve à l'index fourni

Les getters et les setters, ajoute , retire des classes et savoir si il y en a

  • addClass() : ajoute une Class
  • removeClass() : retire une Class
  • toggleClass() : ajoute/retire Class
  • hasClass() : tester l'existence d'une Class
  • is() : tester si on a plusieurs Class

Les événements

La plupart des interaction de l'utilisateur avec la page web peuvent être intercéptées et une action lancée en retour

on peut dire a une fonction de s'executer quand l'utilisateur clique sur un bouton ou bien quand on scrolle la fenêtre

  • mouse :
    click: au click de l'élément
    mouseenter: lorsque la souris passe par dessus
    mouseleave: a la sortit de la zone
  • keyboard :
    keydown: touche enfoncée keyup: touche clavier relachée
  • wndow :
    scroll: difilement de la fenêtre
    resize: redimension de la fênetre
  • form: change: pour les éléments "input" "select" et "textarea", quand l'user change une de ces valeurs
    submit: à l'envoie du form
  • document: DOMContentLoaded : lancé lorsque le document HTML est complètement chargé et analysé sans attendre que les img et le CSS ne le soient
