Les fonctions

Les fonctions permettent de regrouper une série d'instruction pour pouvoir les réutiliser. Quand on a besoin du code d'une série d'instructions, plutôt que de le réécrire à chaque fois toutes les instructionsn, on appellera une fonction par son nom . Cet fonction contient l'enssemble des instructions à executer.

Les instructions

Un script JS est un fichier contenant une ou plusieures instructions, placées les unes à la suite des autres .

Chacune de ces instructions peut-être vue comme un ordre donné au moteur JS, une action que l'on souhaite réaliser a la manière d'une recette de cuisine

  • Beurre
  • Mélangé
  • Verser

Les instructions se suivent et sont éffectuées les unes après les autres, Du haut vers le bas , le browser analyse dans l'order de flux

  • Créer le cadre de la boîte de dialogue
  • Positionner le cadre dans la fenêtre
  • creer un bg semi transparent sur toute la page
  • Placer un bouton avec du text et un événement click
  • Creer une zone de texte qui affichera le texte

Nous avons besoin d'utiliser la fonction alert en lui donnant un argument La fonction éxecutera toute les instructions à chaque fois On peut utiliser cette fonction alert Ce "quelque chose" est un argument attendu par la fonction

Les fonctions prennent souvent des arguments - des bouts de donnée dont elles ont besoin pour fair leur travails . Les arguments - sont placée entres paranthèse, séparées par des virgules s'il yen a plusieurs.

Par exemple la fonction alert() fait apparaîtres une popup mais vous devz donneer une chaîne comme argument pour indiquer à la fonction ce que vous souhaitez écrire dans la fenêtre.
