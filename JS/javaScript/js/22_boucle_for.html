Les boucles

Les instructions FOR & WHILEqui sont 2 types de boucles les plus utilisées en js, ces boucles servent à répéter plusieurs lignes de code

La boucle FOR

FOR prend 3 paramètres encadrés dans des parenthèses, suivi entre accolades des instructions à éxecuter autant de fois que la boucle l'indique . Les paramètres sont:

  • Initialisation : Ici on déclare la variable qui servira de compteur
  • Condition : Tant que cette conditions est vrai la boucle séxécutera
  • Expression finale : Expression qui est éxécutée a chaque fin de boucle
for (initialisation; condition; incrémentation){ -instruction 1; -instruction 2; }
